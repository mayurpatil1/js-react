# Roadmap (summary)

Full interactive version delivered separately as an HTML roadmap/artifact.
This is the plain-text reference for the repo.

## Phase 1 — JavaScript Fundamentals (Days 1–7)
Variables/types, operators/functions, arrays methods, objects/destructuring,
scope/closures/this, promises & async/await, milestone: vanilla JS to-do app.

## Phase 2 — Modern Tooling + TypeScript (Days 8–14)
ES6 modules & classes, error handling/JSON, npm & Vite, TS basics,
generics/unions, typing functions/objects, milestone: port Day 7 app to TS.

## Phase 3 — React Fundamentals (Days 15–28)
JSX/components, props/composition, useState, conditional/list rendering,
forms, useEffect, milestone: React+TS to-do app, custom hooks, Context,
useRef/useMemo/useCallback, React Router, Tailwind, error boundaries,
milestone: multi-page React app.

## Phase 4 — React Ecosystem (Days 29–35)
Data fetching (axios/fetch), TanStack Query, global state (Zustand/Redux
Toolkit), React Hook Form + Zod, auth patterns (JWT), testing (Jest + RTL),
milestone: CRUD app against a mock API.

## Phase 5 — Full-Stack Capstone (Days 36–49)
Design project + schema, build Spring Boot backend, connect React frontend
(CORS), build CRUD pages against the real API, JWT auth end to end,
pagination/search, UI polish, tests, milestone: working full-stack app.

## Phase 6 — Deploy & Portfolio (Days 50–56)
Dockerize backend, deploy backend + frontend, basic CI/CD, documentation,
portfolio/interview prep, next steps (Next.js etc).
