# Day 7 Milestone — Vanilla JS To-Do App

Open `index.html` in a browser (double-click it, no server needed) and try it out.

Reuses everything from Days 1–6:
- closures (Day 5) to keep state private, no globals
- array methods — map/filter/reduce (Day 3) — drive all list logic
- destructuring + spread (Day 4) for immutable-style updates
- async/await (Day 6) via a simulated "sync to server" call
- DOM basics (new): querySelector, event delegation, template-literal rendering

Data persists in the browser's localStorage, so refreshing the page keeps your list.

Stretch exercises are listed as TODOs at the bottom of `app.js`.
